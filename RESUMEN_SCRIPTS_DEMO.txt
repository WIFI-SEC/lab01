💡 PUNTOS CLAVE DE LA DEMO

═══════════════════════════════════════════════════════════════════════════════

1. WPA2 Handshake (DESTACAR ⭐)
   → "4 EAPOL frames = Handshake completo"
   → "Esto es lo que buscan los atacantes para crackear WiFi"
   
   Mostrar: Frame 87, 89, 92, 94
   SSID: Coherer
   Estado: ✓ Completo

2. ARP Spoofing (ALERTA ⚠️)
   → "622 paquetes ARP = posible ataque"
   → "Múltiples MACs por IP = Man-in-the-Middle"
   
   Mostrar: Conteo de requests vs replies
   Indicador: ARP storm detectado

3. HTTP sin HTTPS (SEGURIDAD 🔓)
   → "19 HTTP requests en texto claro"
   → "Contraseñas, cookies, datos visibles"
   
   Mostrar: Hosts contactados
   Advertencia: Usar HTTPS siempre

4. Componentes Crypto (TÉCNICO 🔑)
   → "ANonce y SNonce: números aleatorios del handshake"
   → "MIC: Message Integrity Code para validar"
   
   Mostrar: Primeros 32 caracteres de cada uno
   Uso: Cracking de contraseñas

🎓 EJERCICIOS PROGRESIVOS (10 ejercicios)

📘 NIVEL BÁSICO (30 min cada uno)

  Ejercicio 1: Explorando PCAPs con tshark
    • PCAP: wpa_induction.pcap
    • Aprenden: Comandos básicos, filtros, extracción de SSID/BSSID
    • Verificado: ✅ 4 EAPOL frames, SSID="Coherer"

  Ejercicio 2: Frames WiFi Básicos
    • PCAP: mobile_network_join.pcap
    • Aprenden: Beacon, Probe, Authentication, Association
    • Verificado: ✅ Secuencia completa de conexión

  Ejercicio 3: DHCP y Conexión a Red
    • PCAP: dhcp_traffic.pcap
    • Aprenden: DHCP DORA (Discover, Offer, Request, Ack)
    • Verificado: ✅ 8 mensajes DHCP presentes

📙 NIVEL INTERMEDIO (45 min cada uno)

  Ejercicio 4: WPA2 4-Way Handshake Profundo
    • PCAP: wpa_induction.pcap
    • Aprenden: EAPOL, ANonce, SNonce, MIC, PTK
    • Verificado: ✅ 4 frames completos identificados

  Ejercicio 5: Extracción de Nonces y MIC
    • PCAP: wpa_induction.pcap
    • Aprenden: Crypto keys, PMK, extracción para hashcat
    • Verificado: ✅ ANonce y SNonce extraídos correctamente

  Ejercicio 6: DNS Analysis y Detección de Anomalías
    • PCAP: dns_tunnel.pcap
    • Aprenden: DNS queries, detección de tunneling
    • Verificado: ✅ 30 queries DNS analizables

📕 NIVEL AVANZADO (60 min cada uno)

  Ejercicio 7: ARP Spoofing Detection
    • PCAP: arp_spoofing.pcap
    • Aprenden: Detectar MitM, gratuitous ARP, IDS rules
    • Verificado: ✅ 1000 ARP packets, 500 replies (storm detectado)

  Ejercicio 8: HTTP Traffic Analysis y Captive Portals
    • PCAP: http_captive_portal.cap
    • Aprenden: Session hijacking, cookies, redirects
    • Verificado: ✅ 158 HTTP requests, 6 redirects (302)

  Ejercicio 9: PMKID Attack Simulation
    • PCAP: wpa_induction.pcap
    • Aprenden: PMKID extraction, cracking con aircrack-ng
    • Verificado: ✅ Handshake válido para cracking

🎯 ESCENARIO INTEGRADOR (90-120 min)

  Ejercicio 10: Auditoría Completa de Red WiFi
    • PCAPs: Todos disponibles
    • Aprenden: Análisis forense completo, reporte profesional
    • Objetivo: Integrar todos los conocimientos aprendidos
